# BuddyChat (Frontend)

This is the frontend for **BuddyChat**, a real-time messaging application built with Next.js and Tailwind CSS.

## Features
*   **Premium UI**: Glassmorphism, smooth animations, and interactive design.
*   **Real-time Chat**: Powered by Socket.IO.
*   **Google Auth**: Secure login via Google Identity Services.
*   **Presence**: See who is online instantly.

## Getting Started

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Environment Variables**:
    Copy `.env.local.example` to `.env.local` and add your Google Client ID and Backend URL.

3.  **Run Development Server**:
    ```bash
    npm run dev
    ```

Open [http://localhost:3000](http://localhost:3000) with your browser.
